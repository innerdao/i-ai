# 🤖 Я-ИИ (Ya-II)

Мульти-модельный AI-агент, который использует 7 нейросетей для автоматизации задач.

## Архитектура

| Агент | Модель | Задача |
|-------|--------|--------|
| 🧠 Планировщик | Claude 3.5 | Разбивка задач |
| 🔍 Исследователь | Perplexity | Поиск информации |
| 💻 Кодер | GPT-4o | Написание кода |
| 🎨 Дизайнер | DALL-E 3 | Генерация изображений |
| ✍️ Писатель | Claude | Тексты и доки |
| 📊 Аналитик | GPT-4o | Анализ данных |
| ⚡ Исполнитель | Локально | Запуск и деплой |
